## 🔧 Customization Guide

### Change Color Theme
1. Open `/app/globals.css`
2. Find `:root { ... }`
3. Update color hex codes
4. Save & refresh

### Update Contact Info
1. Open `/lib/constants.ts`
2. Edit `contactInfo` object
3. Update all contact methods
4. Save

### Add Your Projects
1. Open `/lib/constants.ts`
2. Edit `projectsData` array
3. Add your 6 projects
4. Replace images in `/public/images/projects/`

### Change Theme to Alternative
1. See `/COLOR_THEMES.md` for all 8 themes
2. Copy color codes
3. Paste in `app/globals.css`
4. Done!
