1️⃣  START DEVELOPMENT SERVER:
    pnpm dev
    → Open http://localhost:3000

2️⃣  UPDATE YOUR PROJECTS:
    → Open: lib/constants.ts
    → Find: export const projectsData = [
    → Edit: title, description, technologies, metrics, links

3️⃣  CHANGE COLORS:
    → Open: app/globals.css
    → Find: :root { --primary: #3b82f6; ... }
    → Edit: Change hex colors

4️⃣  UPDATE YOUR SKILLS:
    → Open: lib/constants.ts
    → Find: export const skillsData = [
    → Edit: Add/remove skill categories and skills

5️⃣  ADD YOUR EXPERIENCE:
    → Open: lib/constants.ts
    → Find: export const experienceData = [
    → Edit: company, role, period, achievements

6️⃣  ADD TESTIMONIALS:
    → Open: lib/constants.ts
    → Find: export const testimonialsData = [
    → Edit: name, role, content, rating

7️⃣  BUILD FOR PRODUCTION:
    pnpm build

8️⃣  DEPLOY TO VERCEL:
    vercel deploy
    Or push to GitHub and connect Vercel


╔══════════════════════════════════════════════════════════════════════════════╗
║                         WHAT CAN YOU CUSTOMIZE?                              ║
╚══════════════════════════════════════════════════════════════════════════════╝

✅ CAN EASILY CUSTOMIZE:
   • Your projects, skills, experience (in lib/constants.ts)
   • Colors & gradients (in app/globals.css)
   • Page title & metadata (in app/layout.tsx)
   • Navigation links (in lib/constants.ts)
   • Social media links (in lib/constants.ts)
   • Project images (replace in public/images/projects/)
   • Font sizes & spacing (Tailwind classes in components)
   • Animation speeds (in styles/animations.css)
   • Button text & links (in components)

⚠️  ADVANCED CUSTOMIZATIONS (Need React knowledge):
   • Add new sections (create in components/sections/)
   • Create new components (add to components/)
   • Modify animations (edit styles/animations.css)
   • Add new gradients (edit styles/gradients.css)
   • Change layout structure (edit components)

❌ AVOID EDITING:
   • Core animation files (unless you know CSS well)
   • Animation component logic
   • Core library functions
   • TypeScript type definitions

╔══════════════════════════════════════════════════════════════════════════════╗
║                             SUPPORT & HELP                                    ║
╚══════════════════════════════════════════════════════════════════════════════╝

If something isn't working:
  1. Check the browser console for errors (F12)
  2. Restart dev server: pnpm dev
  3. Read the relevant documentation file
  4. Check component code for comments
  5. Review Next.js & Tailwind CSS docs

Common issues:
  ❌ Images not showing?
     → Check file path in constants.ts matches public folder
  ❌ Colors not changing?
     → Clear browser cache and refresh
  ❌ Animations not working?
     → Check browser support, try different browser
  ❌ Build errors?
     → Run pnpm install, then pnpm build
